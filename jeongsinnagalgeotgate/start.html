시작하기

이 문서는 기본적인 정신나갈것같애 언어 사용법에 대해 다룹니다.

저장 공간

  • 스택
  • 큐
  • 태권도장

명령어 조각

정신, 점심

  • 이어지는 명령이 것일 때
    • 정신: 이어지는 명령에서 스택을 사용합니다.
    • 점심: 이어지는 명령에서 큐를 사용합니다.
  • 이어지는 명령이 거일 때
    • 정신: 이어지는 명령에서 입출력을 사용합니다.
    • 점심: 이어지는 명령에서 태권도장에 보관된 값에 대한 연산을 합니다.

나갈, 나가서먹을

  • 이어지는 명령이 것일 때
    • 나갈: 사용하는 저장공간에 태권도장에 보관된 값을 추가합니다.
    • 나가서먹을: 사용하는 저장공간에서 빼낸 값을 태권도장에 보관합니다. 원래 있던 값은 사라집니다.

  • 이어지는 명령이 거일 때

    • 앞선 명령이 정신이었을 때

      나갈: 사용자로부터 입력을 받습니다.

      나가서먹을: 태권도장에 보관되어 있는 값을 유니코드(UTF-8)로 출력합니다.

    • 앞선 명령이 점심이었을 때

      나갈: 에 1을 뺍니다.

      나가서먹을: 태권도장에 보관된 값에 1을 더합니다.

것, 거

것: 앞선 명령에서 스택 또는 큐를 사용합니다.

거: 앞선 명령에서 태권도장을 사용합니다.

같애, 같아

같애: 다음 명령으로 넘어갑니다

같아: 명령이 끝난 후 현재 줄을 기준으로 태권도장에 보관된 값만큼 줄을 이동합니다

  • 태권도장에 보관된 값이 -1일때: 전 줄의 처음 명령으로 이동
  • 태권도장에 보관된 값이 0일때: 현재 줄의 처음 명령으로 이동
  • 태권도장에 보관된 값이 1일때: 다음 줄의 처음 명령으로 이동

?

  • 앞선 명령이 정신이었을 때

    스택에서 빼낸 값을 태권도장에 저장된 값와 비교해 같으면 1, 다르면 0을 스택에 넣습니다

  • 앞선 명령이 점심이었을 때

    큐에서 빼낸 값을 태권도장에 저장된 값와 비교해 같으면 1, 다르면 0을 큐에 넣습니다

.

태권도장에 0을 보관합니다

문장부호는 문장 맨 처음에 사용할 수 없습니다

이외의 명령은 에러를 뱉어냅니다!
